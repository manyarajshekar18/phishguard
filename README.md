# PhishGuard

Advanced phishing detection and URL security analysis web application.

## Features

- URL phishing detection using Gemini AI
- Modern React frontend with cyber-themed UI
- Flask backend API
- Responsive design

## Setup

1. Clone the repository
2. Install backend dependencies: `pip install flask requests`
3. Set GOOGLE_API_KEY environment variable
4. Run the backend: `python app.py`
5. Open http://localhost:5000 in your browser

## Technologies

- Frontend: React, TypeScript, Tailwind CSS
- Backend: Python, Flask
- AI: Google Gemini
